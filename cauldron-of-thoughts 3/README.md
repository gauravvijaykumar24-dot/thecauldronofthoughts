Place your images in the correct folders and name them as above. Recommended sizes:

|-------|-----------------|-------|
| `hero/hero-1.jpg` | 1920 × 1080 px | Landscape, high quality |
| `hero/hero-2.jpg` | 1920 × 1080 px | Landscape, high quality |
| `hero/hero-3.jpg` | 1920 × 1080 px | Landscape, high quality |
| `author/author.jpg` | 440 × 440 px | Square, portrait |
| `pillars/military.jpg` | 800 × 600 px | Landscape |
| `pillars/horizons.jpg` | 800 × 600 px | Landscape |
| `pillars/geopolitics.jpg` | 800 × 600 px | Landscape |
| `pillars/shivaji.jpg` | 800 × 600 px | Landscape |

> **Note:** If images are missing, graceful fallbacks (gradient backgrounds and placeholder icons) are shown automatically.

## 🚀 Deploying to GitHub Pages

1. Push all files to a GitHub repository (e.g., `cauldron-of-thoughts`)
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at:
   `https://yourusername.github.io/cauldron-of-thoughts/`

## ✏️ Customising Social Media Links

In `index.html`, find the `social-links` section in the footer and replace the `href="#"` or placeholder URLs with your actual profiles:

```html
<a href="https://twitter.com/YOUR_HANDLE" ...>
<a href="https://linkedin.com/in/YOUR_PROFILE" ...>
<a href="https://youtube.com/@YOUR_CHANNEL" ...>
<a href="https://instagram.com/YOUR_HANDLE" ...>
<a href="https://facebook.com/YOUR_PAGE" ...>
```

## 🎨 Design Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--navy` | `#0D1B2A` | Navbar, footer background |
| `--gold` | `#C9A84C` | Accents, borders, hover states |
| `--crimson` | `#8B1A1A` | Active state accents |
| `--cream` | `#F8F6F1` | Page background |
| `--slate` | `#4A5568` | Body text |

Change these in `assets/css/style.css` under `:root` to retheme the entire site.
